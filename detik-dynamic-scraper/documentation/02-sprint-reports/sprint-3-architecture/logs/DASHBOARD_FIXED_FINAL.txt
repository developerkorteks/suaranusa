Problem: Logger import path conflicts between dashboard and src modules

Fix: Add fallback dummy logger in database.py

Code:
```python
try:
    from utils.logger import setup_logger
    logger = setup_logger(__name__)
except ImportError:
    # Fallback: create dummy logger for dashboard compatibility
    class DummyLogger:
        def info(self, msg): print(f"INFO: {msg}")
        def warning(self, msg): print(f"WARNING: {msg}")
        def error(self, msg): print(f"ERROR: {msg}")
        def debug(self, msg): pass
    logger = DummyLogger()
```

Why it works:
- If logger can be imported (direct use): Uses real logger
- If logger cannot be imported (dashboard): Uses fallback dummy logger
- Dashboard doesn't break, just uses print instead
